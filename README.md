# Grid Maker

Free browser-based image grid tools for splitting social media posts, adding
drawing grids, making crochet grids, and previewing pixel grids.

## Stack

- TanStack Start
- React 19
- Tailwind CSS v4
- Cloudflare Workers

## Development

```bash
pnpm install
pnpm run dev
```

The local app runs at http://localhost:3000.

## Production

Production is configured for:

```txt
https://anygridmaker.com
```

Deploy with:

```bash
pnpm run deploy
```
